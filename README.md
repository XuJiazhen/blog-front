# 个人博客系统：FRONT

介绍：目前只是第一阶段开发完成，主要实现功能如下：

- 从后台拉取文章并展示列表
- 根据文章 ID 拉取指定文章并展示详情
- 只实现了 PC 端的页面适配

## 使用技术

- NuxtJS + Vue-Router + Vuex + Axios
- Marked + Highlight.js

```bash

$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
